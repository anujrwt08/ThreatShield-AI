# 🛡️ ThreatShield AI — Enterprise-Grade Neural Security

**ThreatShield AI** is a next-generation cybersecurity platform designed for the modern enterprise. It combines AI-native threat detection with a high-performance, reactive dashboard to neutralize perimeter breaches before they occur.

---

## 🚀 Overview

Initially conceived as a design prototype in **Google Stitch**, ThreatShield AI has been fully migrated into a robust, full-stack **Next.js 16** application. The platform provides a unified interface for incident logging, threat simulation, and autonomous agent management.

## ✨ Key Features

### 📊 Reactive Security Dashboard (Logs)
- **Real-time Incident Tracking:** Monitor incoming threats with live severity grading (Critical, Warning, Info).
- **Instant Mitigation:** One-click protocols for quarantining malicious traffic or viewing deep-dive forensic data.
- **System Health Metrics:** Live visualization of Neural Load, Database Integrity, and System Posture.

### 🧪 AI Threat Analyzer (Simulation Sandbox)
- **Zero-Day Simulation:** A functional sandbox where users can paste hashes or URLs to see the AI neutralization engine in action.
- **Secure Verification:** High-speed scanning with simulated results used to train operators on threat identification.

### 🤖 Neural Agent Fleet (`/agents`)
- **Autonomous Nodes:** Deploy and manage a global fleet of security agents (Sentinel Alpha, Neural Node-02, etc.).
- **Live Fleet Control:** Interactive toggles to activate/deactivate perimeter nodes with real-time performance feedback.

### 🏛️ Complete Marketing Ecosystem
- **Dedicated Solutions:** Detailed breakdowns of Cloud, Endpoint, and Network defense modules.
- **Tiered Pricing:** Transparent subscription plans from Startup to Enterprise tiers.

## 🏗️ How It Was Built

1.  **Migration & Architecture:** Rebuilt the original Stitch HTML structure into modular React components using the Next.js App Router for optimal performance and SEO.
2.  **Design System:** Implemented a custom **Cyberpunk Glass-morphism** theme using Tailwind v4. This includes custom CSS utilities for neon glows and micro-animations.
3.  **Interactivity:** Built with a "Function First" philosophy. Every button and simulation uses React state and effects to mimic professional security software.
4.  **Mock Data Layer:** Developed robust internal APIs to serve incident logs, system metrics, and agent fleet data, ensuring a "live" feel without needing a persistent database setup for the demo.

## 🏁 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-repo/threatshield-ai.git
    cd threatshield-ai
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the dashboard:**
    Navigate to [http://localhost:3000](http://localhost:3000) to begin the protocol.
